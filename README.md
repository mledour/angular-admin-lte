## Installation

To install this library, run:

```bash
$ npm install angular-loading-page --save
```
## Install demo
```bash
$ git clone git://github.com/mika-el/angular-admin-lte
$ cd angular-admin-lte
$ npm install
```

## Run demo
```bash
$ ng serve
```

## Build demo
```bash
$ ng build --prod
```

## Build dist
```bash
$ npm run build:lib
```
